TAG BASE DEL DOCUMENTO HTML

 

  • <HTML></HTML>        (inizio e fine del file, tag obbligatorio)
  • <HEAD></HEAD>      (descrizioni varie; come il titolo, Meta Name)
  • <TITLE></TITLE>        (nome del documento; contenuto in Head)
  • <LINK REL=STYLESHEET HREF="url" TYPE="text/css">  (riferimento a foglio di stile, generalmente contenuto in Head)
  • <STYLE TYPE="text/css">
    <!--
    BODY {font-family: Arial; font-size: 12pt; font-style:italic}
    P{margin-left: 30px; margin-right: 30px; margin-top: 10px}
    -->
    </STYLE>
  • <SCRIPT>...
    ...istruzioni...
    </SCRIPT>

            Questo tag contiene delle istruzioni di script da far eseguire al browser. Attributi: 

            LANGUAGE="linguaggio" specifica in quale linguaggio è stato composto lo script, come VBScript oppure JavaScript.

            SRC="..." specifica il percorso per raggiungere il file oggetto.

  • <BODY></BODY>       (corpo del file)
