Istanze di Ente

 

L'Istanza di DataBase è l'insieme delle informazioni registrate nella Base Dati relativamente ad uno specifico Progetto, per la gestione di un Ente e per l'identificazione di uno User del DataBase e della password utilizzata.

In fase di installazione del Progetto viene predisposta una Istanza con lo stesso nome del Progetto, ma è possibile definire nuove Istanze con la fase Progetti presente all'interno del menu Archivio.

Possono essere definite anche più Istanze per lo stesso ente su diversi User DB, per gestire più di una Base Dati, o più Istanze per enti diversi sullo stesso User DB, per l'accesso multiplo alle stesse informazioni con Personalizzazioni a livello di ente.

Per la modifica delle Istanze è necessario utilizzare la funzione relativa alla definizione dei Progetti.
