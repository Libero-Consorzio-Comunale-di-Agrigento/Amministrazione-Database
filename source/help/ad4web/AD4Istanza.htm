Istanza

 

Pagina che consente di inserire o modificare un'Istanza di DataBase predisposta nella Base Dati del sistema.

 

 

Dati

Istanza

Codice e la descrizione dell'istanza.

Ente

Indica l'ente associato all'istanza.

Dislocazione

Rappresenta la dislocazione dei file componenti il prodotto.

Temporanea

Rappresenta, invece, la dislocazione de i file temporanei dell'applicazione.

Lingua

Rappresenta la lingua scelta dall'utente per la personalizzazione dell'applicazione; l'attributo è messo a disposizione dei soli applicativi che ne prevedono la gestione.

DB User

Individua l'utente di connessione al database. Accanto al campo è presente un pulsante che consente di effettuare il cambio della password.

Oracle Link

Stringa di connessione SQL*Net

Database Link

Stringa di connessione in formato host:port:sid o nel formato previsto dagli applicativi che ne prevedono la gestione.

Database Driver

 

Driver da utilizzare per la connessione a Database Link.

Servizio

È un campo di servizio messo a disposizione dei soli applicativi che ne prevedono la gestione.

Note

È presente anche un campo Note che consente di inserire eventuali note aggiuntive, fino ad un massimo di 2000 caratteri.

Versione ed Installazione

Visualizzano rispettivamente la versione del programma installato sul server ed i componenti installati.

 

Funzioni Contestuali

Caratteristiche

Pagina che consente di definire le caratteristiche dell'accesso all'istanza selezionata.

Abilitazioni

Visualizza la situazione completa degli utenti che hanno abilitazione di accesso all'istanza selezionata.

Registro

Pagina che consente di visualizzare e modificare il registro di configurazione dello user di database associato all'istanza.
